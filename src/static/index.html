Documentation

Base URL
http://localhost:8000/api/v1
Book Object Structure
{
  "id": 1,
  "title": "string (1-255 characters)",
  "author": "string (1-255 characters)",
  "description": "string or null (max 1000 characters)",
  "status": "string (1-50 characters, default: 'available')",
  "created_at": "ISO 8601 datetime"
}
Error Responses
Status Code Description
400 Bad Request - Invalid input data
404 Not Found - Book with specified ID doesn't exist
500 Internal Server Error - Server-side issue
Usage Examples

Get all books using curl:

curl -X GET "http://localhost:8000/api/v1/books/"

Create a book using curl:

curl -X POST "http://localhost:8000/api/v1/books/" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "The Great Gatsby",
    "author": "F. Scott Fitzgerald",
    "description": "A classic novel",
    "status": "available"
  }'

Get a specific book:

curl -X GET "http://localhost:8000/api/v1/books/1"

Update a book:

curl -X PATCH "http://localhost:8000/api/v1/books/1" \
  -H "Content-Type: application/json" \
  -d '{"status": "borrowed"}'

Delete a book:

curl -X DELETE "http://localhost:8000/api/v1/books/1"
